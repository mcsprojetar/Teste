Aplicativo PWA completo para gestão de pedidos, clientes e fornecedores.

## ✨ Funcionalidades

- 📝 Cadastro de pedidos com múltiplos produtos
- 👥 Lista de clientes com busca e filtros
- 🏭 Lista para fornecedores com agrupamento automático
- 📊 Dashboard com gráficos e estatísticas
- 📈 Relatórios por período
- 💾 Backup e importação de dados
- 🌙 Modo escuro
- 📱 PWA instalável em Android e iOS
- 🔄 Funcionamento offline

## 📱 Instalar no Celular

### Android (Chrome)
1. Acesse o site no Chrome
2. Toque nos 3 pontos ⋮
3. Selecione "Adicionar à tela inicial"
4. Confirme "Instalar"

### iPhone/iPad (Safari)
1. Acesse o site no Safari
2. Toque no ícone Compartilhar
3. Role e selecione "Adicionar à Tela de Início"
4. Confirme "Adicionar"
